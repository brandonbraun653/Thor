31  /** A high level, basic serial interface for rapid prototyping. This is essentially a wrapper over the existing
32  * Thor::Peripheral::UART::UARTClass and Thor::Peripheral::USART::USARTClass interfaces and implements much of
33  * the same features found there. The goal of this class is to abstract away from the specific UART and USART
34  * peripherals so that the user only has to call 'SerialClass(1)' or 'SerialClass(x)' to generate a serial object
35  * quickly without having to worry about the underlying hardware type.

46  * @param[in] tx_mode Sets the TX mode to Blocking, Interrupt, or DMA from Thor::Definitions::Serial::Modes
