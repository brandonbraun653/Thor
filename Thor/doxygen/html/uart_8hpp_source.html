44  /** A higher level uart interface built ontop of the STM32 HAL that abstracts away most
45  * of the details associated with setup and general usage. It supports both transmission\n
46  * and reception in 3 modes [blocking, interrupt, dma] and does not require that TX and RX
47  * share the same mode for proper operation. In addition, data is copied to an internal buffer\n
48  * for all non-blocking transmissions (IT/DMA) so that the user doesn't have to worry about
49  * destroyed or mutable data.
50  *
51  * @note If using FreeRTOS, the class is threadsafe and allows multiple sources to write and
52  * read on a class object up to an internal buffer limit defined by Thor::Definitions::Serial::UART_BUFFER_SIZE
53  **/
