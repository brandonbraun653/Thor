Detailed Description

A higher level uart interface built ontop of the STM32 HAL that abstracts away most of the details associated with setup and general usage.

Note
If using FreeRTOS, the class is threadsafe and allows multiple sources to write and read on a class object up to an internal buffer limit defined by Thor::Definitions::Serial::UART_BUFFER_SIZE

A factory method to create a new UARTClass object.

This method intentionally replaces the typical constructor due to the need to register
the shared_ptr with a static_vector that allows runtime deduction of which class to call
inside of an ISR. This is done for simplicity.
